# ORKA Heating & Plumbing Website

## Run locally
1. Install Node.js 20.19+ or 22.12+
2. Open this folder in terminal
3. Run:

```bash
npm install
npm run dev
```

## Before going live
- Replace `your-form-id` in `src/App.tsx` with your real Formspree ID.
- Turn on Formspree email notifications and autoresponder.
- Push this folder to GitHub.
- Import the repo into Vercel and deploy.

## Main files
- `src/App.tsx` - website
- `public/logo.jpg` - company logo
- `public/jobs/` - gallery images
